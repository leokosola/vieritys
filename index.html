Vieritystarina internetistä

Internetin toiminta

Jotta laitteen voi kytkeä internet-verkkoon, sen tarvitsee pyytää itselleen yksilöllinen tunniste eli IP-osoite. Tunnisteen saa automaattisella DHCP-pyynnöllä paikallisverkon reitittimestä.
IP = Internet protocol

Reititin vastaa pyyntöön lähettämällä IP-osoitteen. Lisäksi reititin lähettää oman osoitteensa ja lähimmän DNS-palvelimen osoitteen tulevia pyyntöjä varten.
DNS = Domain Name Server
